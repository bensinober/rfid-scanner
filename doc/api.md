# API

The server exposes APIs for querying the status of shelves, items or devices.

### `/json/shelves/`
```
GET /json/shelves/
{
   at: {
      epoch: 1517398770.60357,
      iso8601: "2018-01-31T11:39:30"
   },
   results: [
      {
         loc: "x.y.z",
         tags: "123"
      },
      {
         loc: "hutl.stack.foo",
         tags: "2"
      },
      ...
   ]
}
```
